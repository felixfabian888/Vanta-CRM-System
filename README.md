# CRM System

Hi there! 👋 This is a Customer Relationship Management (CRM) application I am building from scratch using **Flutter**. 

My goal with this project is to build a clean, modern, and truly cross-platform app that works beautifully on both mobile phones and web browsers.

> **Current Status:** 🚧 Actively being built. I'm currently working on the core features, but the foundation (like the responsive login UI) is ready to go.

---

## 💡 Why I'm Building This

I wanted to move beyond simple tutorial apps and build something that mimics real-world software. This project is my playground for:
- Writing clean, organized, and reusable Flutter code.
- Mastering responsive design (so it adapts perfectly to desktop and mobile).
- Learning how to structure a larger app with proper folders and state management.
- Experimenting with modern UI trends, like glassmorphism and smooth gradients.

---

## ✨ What's in it right now?

- **Responsive Login Page:** A beautiful, glassmorphism-style login screen that automatically changes its layout depending on whether you're on a phone or a computer.
- **Clean Architecture:** I've separated the UI into small, reusable widgets (like custom text fields and glass cards) so the code is easy to read and maintain.
- **Form Validation:** Real-time checking for emails and passwords before submitting.

*(More features like dashboards, contact lists, and data tracking are coming soon!)*

---

## 🛠️ Tech I'm Using

- **Framework:** Flutter
- **Language:** Dart
- **Design:** Custom UI with glassmorphism effects (no heavy third-party UI libraries, I like building it myself to learn)
- **Version Control:** Git & GitHub
